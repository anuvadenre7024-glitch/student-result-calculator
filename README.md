# Student Result Calculator

A Python-based application that calculates a student's total marks, percentage, and grade based on user input.

## Features

- Accepts student details and marks
- Calculates total marks
- Computes percentage
- Generates grade automatically

## Technologies Used

- Python

## How to Run

1. Run the Python file.
2. Enter student name.
3. Enter marks.
4. View result and grade.
